一、安装nodejs插件

  1. 下载：nodejs，选择 LTS x64 版本；

  2. 安装一直点下一步，路径可自定义，Automatically install the nessary tools 选项不要勾选，否则会下载耗时长；

  3. 测试是否安装成功：win+r，cmd，终端命令行输入 node -v 和 npm -v，分别显示nodejs和npm版本说明安装成功；

  • 注意：新版nodejs自带npm，随nodejs一起安装，其作用是对nodejs依赖的包进行管理，也可以理解为用来安装/卸载nodejs需要装的东西；
